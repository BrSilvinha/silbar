# SILBAR — Cómo ejecutar

## Requisitos
- Node.js instalado (v18 o superior)

## Pasos para correr el proyecto

### 1. Abrir DOS terminales

**Terminal 1 — Servidor (backend):**
```
cd C:\Users\jhami\OneDrive\Documentos\proyectos\silbar\server
npm start
```

**Terminal 2 — Cliente (frontend):**
```
cd C:\Users\jhami\OneDrive\Documentos\proyectos\silbar\client
npm run dev
```

### 2. Abrir en el navegador
```
http://localhost:5173
```
